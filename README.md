
# Operação

Processamento: Rode o node batch-processor.js para garantir que o integrity_map.json e os vídeos HLS estejam gerados.

Execução: Inicie o servidor(CMD) com node server.js. O console deve exibir: 🚀 Rede Segura Ativa em https://localhost.
